HUB

Speak or type, then Send to G2. Chips add phrases. On glasses, controls are a narrow list on the left — swipe to move the highlight, tap to run — so the sign uses the large area on the right. Clear (or double-tap from the main nav slide view) returns to the idle screen. From Phrases, double-tap first returns to main nav + idle; double-tap again on that idle screen opens the system exit, or tap Exit.

Glasses controls & shortcuts

Speak then Send, or Ctrl+Enter. On G2: Prev · Next · Replay · Clear · Phrases · Exit. Double-tap the list: from the main nav while a deck is showing, goes to idle home; from Phrases, goes to main nav + idle first; then double-tap again on idle home opens the system exit (same as Exit). The list is a slim left column (swipe / tap). Each Send replaces the deck. After send, Alt+← / → (outside the field) steps preview like the glasses.

Aid only — not a certified interpreter or emergency service. Text stays here until you send. Speech may use your device recognizer.
