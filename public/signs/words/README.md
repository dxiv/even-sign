# `words/*.png`

Built with `npm run build:words` from `src/signSlides.ts` + `alphabet/` + `numbers/`. Needs `npm run build:signs` first, or use `npm run build:signs:all`. Long titles use up to **four rows** and a **minimum letter height** so strips stay readable on G2 before any final shrink.

Drop your own `hello.png` etc. here to override; re-run `build:words` only if you want to regenerate from letters.
